# WOFES AI v2 — Космический AI-чат 🌌

**WOFES AI v2** — это стильный AI-помощник с космическим дизайном, регистрацией, историей сообщений и ротацией API-ключей.

## 🚀 Возможности

- ✨ **Космический интерфейс** — анимированные звёзды, туманности, стеклянные карточки
- 🔐 **Регистрация и вход** — аккаунты сохраняются в браузере
- 💬 **Умный чат** — поддерживает несколько диалогов, историю сообщений
- 🔑 **Ротация ключей** — 7 API-ключей для стабильной работы
- 📱 **Адаптивный дизайн** — отлично работает на телефонах и планшетах
- 🤖 **Поддержка моделей** — DeepSeek, Gemini, Llama через OpenRouter

## 🛠️ Технологии

- HTML5, CSS3, JavaScript (vanilla)
- OpenRouter API
- localStorage для хранения данных
- Адаптивная вёрстка (Flexbox/Grid)

## 📦 Установка и запуск

1. Скачай все файлы репозитория
2. Открой `index.html` в браузере
3. Зарегистрируйся или войди
4. Начни общение с AI!

Или просто перейди на сайт который уже запущен
